# 部署说明

## 环境要求
- Python 3.8+
- Node.js 16+
- SQLite (默认) 或 MySQL/PostgreSQL

## 后端部署
1. 安装依赖: `pip install -r requirements.txt`
2. 迁移数据库: `python manage.py migrate`
3. 收集静态文件: `python manage.py collectstatic`
4. 启动服务: `gunicorn config.wsgi:application`

## 前端部署
1. 安装依赖: `npm install`
2. 构建: `npm run build`
3. 将 `dist` 目录部署到 Nginx 或静态文件服务器。

## Nginx 配置示例
```nginx
location /api {
    proxy_pass http://localhost:8000;
}

location / {
    root /path/to/dist;
    try_files $uri $uri/ /index.html;
}
```
